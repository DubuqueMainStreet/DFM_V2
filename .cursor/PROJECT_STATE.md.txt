## Plugin & Tool Usage

### Context7 (MCP: `plugin-context7-plugin-context7`)
Use when working with Wix Velo, wix-data, wix-crm-backend, or Leaflet.js and you need accurate, up-to-date API docs.

**Flow (always two steps):**
1. `resolve-library-id` — Pass the library name (e.g., "wix-data") to get its Context7 ID
2. `query-docs` — Pass the resolved ID + your specific question to get docs and code examples

**When to use:**
- Implementing or debugging any Wix API call (wix-data, wix-crm-backend, wix-auth, triggered emails)
- Working with Leaflet.js map features
- Unsure about function signatures, parameters, or return types
- Max 3 calls per question

**Pre-resolved library names for this project:**
- `wix-data` — CMS queries, insert, update, remove
- `wix-crm-backend` — Contact creation, triggered emails, elevate
- `wix-auth` — elevate() for backend permissions
- `leaflet` — Map rendering, markers, popups, layers
